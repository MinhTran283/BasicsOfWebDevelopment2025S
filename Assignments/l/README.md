# UrbanRide — Final Website (Task L)

This folder contains the final refined version of my rental website,
combining work from Tasks I, J, and K into a coherent three-page site.

## Pages included
- index.html — Landing page
- catalog.html — Rental catalog
- order.html — Rental order form

## Improvements made
- Unified navigation, header, and footer across all pages
- Consistent color palette, typography, spacing, and buttons
- Improved semantic HTML structure and heading hierarchy
- Fixed validation issues reported in earlier tasks
- Added required example.jpg to catalog page
- Improved form accessibility with labels, fieldsets, and validation
- Ensured responsive layout on mobile, tablet, and desktop

## Validation
All pages were checked using the W3C HTML Validator and corrected.
